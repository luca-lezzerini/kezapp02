Scenario Type
Basic path for All Basic Path
Details:
User selects all in the combo, then the grids of expired dates and next expiration dates displays the expired dates and next expiration dates sorted DESC by expiration date.<br/>
Basic path for person Alternate
Details:
User selects person in the combo, then selects one person in the selector grid (see UI): the grids of expired dates and next expiration dates displays the expired dates and next expiration dates sorted DESC by expiration date and filtered by the selected person. If no person is selected, only expiration dates about persons are displayed.<br/><br/>
Basic path for asset Alternate
Details:
User selects asset in the combo, then selects one assetin the selector grid (see UI): the grids of expired dates and next expiration dates displays the expired dates and next expiration dates sorted DESC by expiration date and filtered by the selected asset. If no asset is selected, only expiration dates about persons are displayed.<br/><br/>
Basic path for expiration periods Alternate
Details:
User selects expiration period in the combo, then selects one expiration period in the selector grid (see UI): the grids of expired dates and next expiration dates displays the expired dates and next expiration dates sorted DESC by expiration date and filtered by the selected expiration period. If no expiration period is selected,it works like All case.<br/><br/>
