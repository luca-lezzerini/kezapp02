This use case allows the user to select one or more assets and to associate them to one or more expiration periods.<br/>The user can also see which expiration periods have already been associated to a selected asset or vice versa.<br/>When creating a new ExpirationAssociation association, the lastEventDate field must be left empty.<br/>
