Double clicking on one row opens a modal when the user can see old expiration date and person or asset information, the list of changes (ForcedLog) done for that date and then can input: new expiration date and reason.<br/>Confirming, saves the new expiration date in both ExpirationDate record and new created ForcedLog record and closes the modal.<br/>On each row, in the rightmost column, a Show History button is available.<br/>If the Show History button is clicked, the modal is opened with no input fields and no buttons and it is used to display the forced log records only related to that row.<br/>
