This entity will represent the exceptional availability o unavailability of a resource or of a resource category.<br/>This means that a resource (or category) has been set (through Calendar) as available in some time but this has to be changed (without changing the Calendar but the resource availability only).<br/>This record will override all other availabilities in case of conflict.<br/>In case of conflict between different ResoourceAvailabilityChange, the unavailability will prevail.<br/><br/>
